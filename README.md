# 🌐 Arni Johry — Portfolio Website

This is my personal developer portfolio built to showcase my skills, projects, experience, and achievements as a Computer Science student focusing on Cloud, DevOps, and Backend Development.

---

## 🚀 Live Website
👉 https://arni.live

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS

### Deployment
- AWS Amplify (CI/CD + Hosting)
- CloudFront (handled automatically by Amplify)

### Domain
- Custom domain purchased from Name.com → `arni.live`

## 📌 Features

- 🎯 About Me section with introduction and background
- 💡 Skills section showcasing technologies and tools
- 💼 Experience timeline (internships & work experience)
- 🏆 Achievements & certifications
- 📂 Projects with GitHub integration
- 📬 Contact form (Email, Phone, Location, Social links)
